CLOUD PROJECTS

Prompt Risk Analsis Engine

Developed a full-stack Hybrid AI system to analyze and score LLM prompts for clarity, specificity, and hallucination risk. Fine-tuned a custom multi-head PyTorch Transformer to provide base analysis, complemented by novel Explainable AI (XAI) techniques that generate a token-by-token explanatory trace. Architected with a React/TypeScript frontend, a FastAPI backend, and a foundational MLOps CI/CD pipeline using GitHub Actions.

On-Demand LLM Detector

Fine-tuned a transformer model for LLM text detection, achieving 99.8% F1 Score, reflecting excelleent precision and recall. Deployed with custom zero-to-one serverless scaling. Set a Lambda trigger to spin up ECS tasks behind a load balancer on-demand, with containers self-terminating after inactivity periods to eliminate idle infrastructure costs.

Is it a Dog? Serverless API App

Implemented a serverless image recognition API using AWS Lambda, API Gateway, and S3, leveraging Amazon Rekognition to identify dogs within uploaded images.

Automated Email Triage Workflow

Designed and implemented a personal productivity workflow using the open-source automation tool N8N, which leverages Gmail APIs and conditional logic to filter and categorize incoming messages in real time. DynamoDB for data persistence.
